# 贡献指南

感谢你愿意一起完善 `EasyNote`。

这个仓库当前以现有主线作为正式发布主线，所以提交改动时请优先保证现有正式版行为不被破坏。

## 建议流程

1. 先阅读 [README.md](./README.md)，确认当前项目目标和安装方式。
2. 尽量做小步、可回滚的改动。
3. 优先复用现有实现，不轻易增加新依赖。
4. 提交前至少完成一次本地构建验证。

## 本地验证

```powershell
dotnet restore .\easy-note-wpf.sln
dotnet build .\easy-note-wpf.sln
```

如果你的改动影响发布或安装流程，再补充执行：

```powershell
.\build-windows-installer.ps1
```

如果你的改动影响 Windows 安装包行为，建议再补充验证：

- 安装器是否默认落到 `C:\Program Files\EasyNote`
- 安装器是否正常请求管理员权限
- 安装完成后是否可直接启动，且不再弹出 .NET 运行时缺失提示

## 提交建议

- 优先修复明确问题，避免顺手改动过大
- 涉及桌面嵌入、托盘、自启动或全局快捷键时，请在说明里写清实际验证结果
- 如果发现仍无法确认的行为，请把风险写清楚，方便后续继续接力

## Issue / PR 建议内容

- 问题现象或目标
- 复现步骤
- 预期行为
- 实际行为
- 验证环境

欢迎继续把这个项目做得更稳定、更易用。
